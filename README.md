# Overview

**Treeliable: Smash the Ore** is an addon for Treeliable, that adds 3x3 veinmining.
### Dependecies
- [Treeliable](https://modrinth.com/mod/treeliable)
- [Cloth Config](https://modrinth.com/mod/cloth-config)
## Features
- **Veinmining:** 3x3 blocks ore mining, that only works with pickaxes. *Can be set to break blocks by layers, or by blocks.*
